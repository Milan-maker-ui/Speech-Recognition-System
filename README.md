# Speech Recognition System

A Python-based Speech Recognition System that records audio from the microphone and converts speech into text using the `speech_recognition` library.

---

## Features

* Record audio from microphone
* Save recordings automatically
* Convert speech to text
* Adjustable recognition sensitivity
* Error handling for recording and transcription

---

# Example Workflow

1. Start recording
2. Speak into microphone
3. Audio gets saved in `recordings/`
4. Speech is converted into text
5. Display transcription result

---
